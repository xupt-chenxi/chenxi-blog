    # 申请友链

    一点点要求：

    • 您的网站要有独立域名，且支持 HTTPS 访问
    • 能以较快速度访问到您的网站
    • 确保您的网站内容健康，且多为原创，转载需注明出处
    • 需将本站也设为友链

    如您满足以上要求，就请去👉留言发起友链申请吧

    申请格式：

    昵称（可选）
站点URL（https://）
站点名称
主人头像（https://）
站点描述（一句话即可）

    例：

    昵称：chenxi
站点URL：https://chenxi-space.com
站点名称：chenxi-space
主人头像：https://pic.imgdb.cn/item/65919713c458853aef5343e9.jpg
站点描述：行到水穷处，坐看云起时

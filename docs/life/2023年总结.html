              2023：有所改变的一年

              在以往，我从未给自己做过年度总结，可从2023年开始，我想开始做这件事
              从2023开始也挺好的，因为2023真的有那么一点不一样

              不过在开始总结前，先允许我聊一聊最近吧

              # 近况

              要说到近期做的事，无非也就是 建站，学Java，以及 备战期末考试

              建站

              建这个我自己的小站是我在近期耗时最长的一件事，其实从某种意义上来说，我不该花太多时间在这上面
              而是应该继续投入到 Java的学习当中（毕竟是想要在下一学期找实习的）
              但我还是去做了，并乐此不疲，因为对我而言这是一件很有意义的事，且是很早以前就想做的事，我只是在最近将其付诸实践了而已

              我发现我在某些事上有异常的执着
              或许我表现得像是已经忘记的样子，但其实我内心一直惦记着

              不管怎样，在今天，1月2日，2024的第二天，这件事也划上了句号
              成品就在你的眼前，你觉得怎样

              学 Java 在下文2023总结里会提，这里就不说了

              备战期末考试

              西邮的期末考试，于我而言一直都是一件很无聊的事
              令人作呕的考试风气，毫不作为的监考老师
              我们专业估计作弊率估计能达到 70%，考试结束后，身边人讨论的或许不是题的难易度，而是你抄到了吗，令人好笑却又无奈

              而我呢？也并非是什么上课好好听，然后顺理成章考出高分的学生，无非也就是考前靠老师划的重点，突击一下，以考过为目标罢了，但光是不作弊这点，在西邮，某种意义上来说都能算是一股清流了

              在期末考试期间，我右眼还发炎了（病毒性结膜炎），眼睛肿得都快睁不开了，又疼又痒，泪流不止，但因为要突击期末考试，又不能停止学习，也曾想过摆烂，但终究还是坚持了下来

              好在，用药以后，今天已经好得差不多了
              以后一定要好好保护自己的眼睛

              接下来就进入2023总结吧
              词穷，所以，我对2023的总结词就是，有所改变

              # 2023：有所改变的一年

              # 疫情结束

              还记得那是在我去楼下广场做核酸的时候，核酸的负责人，说着，“不用做了，不用做了”（大抵是这个意思），当时我还有些懵
              回到家，打开手机就看了放开的消息，当时的感觉真就只有不真实感，因为真的已经封太久了

              但人的适应能力是很强的，就像疫情封锁的时候，人们能照样过活
              疫情解封时，人们也能很快适应
              慢慢的开始摘下口罩，慢慢的回归以往的生活
              从疫情结束到今天，已经一年多了，在此刻回想起解封前的生活，已经有了些朦胧感

              解封后，也是和朋友去了一些地方玩，但我本身也算是比较宅，再加上与我而言，去什么地方玩或许不是重点，重点是和谁去，所以，整体外出频率仍比较低

              不过疫情的解封，却让我去做了一件对我很重要的事：戴牙套

              # 戴牙套

              这件事在我大学入学时就想做了，但一直拖着，主要是两个原因：

              • 害怕
              • 疫情封锁

              于是乎我就一直从大学入学，一直拖到了大二下，也就是疫情解封后
              因为大二下或许是我最后的机会，再拖就没有时间了
              正好疫情结束了，一边想着，冲吧！一边又因为网上看到的失败案例，让我畏畏缩缩，又是纠结了一段时间

              终于，在 2023.5.12 时，去医院进行了初诊

              对于医院的选择，我倒是没有太纠结，出于安全性，不考虑私立，从西安这边的公立医院来看，也就是第四军医大和西交口腔二选一

              最后我选择了西交口腔

              第二次到院时，就是谈方案了

              当时我被告知需要拔4颗智齿，4颗！出于对拔牙的恐惧，我又有点退缩了
              不过终究还是心一横就去拔了，也是在西交口腔拔的，由于医生技术很好，拔牙的过程很顺利（我是一次拔两颗，分两次才拔完）

              或许是经历了拔牙的痛苦，让我对戴牙套莫名有了信心，想着，拔牙都熬过来了，戴牙套算啥

              于是在 2023.6.15 我去到医院戴上了牙套，我原本对于戴牙套的感觉就是，可能会有一点不适，然后不能吃硬物，其他可能就没啥问题了，但事实证明我想少了

              戴牙套的过程倒比较顺利，刚上完牙套那会，还并没有什么特别明显的不适感，但难受的远在后面：

              • 两边最后的一颗牙之间要连皮筋（这真的是个技术活，最开始的时候真的怎么都做不好）
              • 两边大牙垫了蓝胶（咬合就只有大牙能咬上，几乎没法吃东西，只能喝粥）
              • 牙内侧用来挂皮筋的钉子卡舌头（更是重量级，舌头一直被这东西划破，那段时间吞口水都像吞针一样疼）
              • 不能吃硬物，饭后必须刷牙（也就是说我几乎没办法去外面吃饭，对于朋友的邀约也只能无奈推掉）


              但随着时间的推移，这些痛苦也就逐渐的减弱，到了今天，我已经完全适应了牙套的生活
              上个月去复诊时，医生说我的矫正效果很不错，已经进入最后调整阶段了

              想到摘掉牙套后要做的事，我想，我应该请朋友们出去吃一顿饭

              # 选择方向

              2023年，对我而言，还比较重要的一件事就是想好了自己所想要走的路

              其实在自己大一大二时，对自己想做什么是很迷茫的，又或者自己根本就没去想过这个问题，大抵的方向倒是有，比如：考一个好研究生或是找一份好工作，但却不够具体，哪个学校的研究生？什么方向的工作？我是没有去具体想过的，这么说也不太对，我倒一直有从事后端方向工作的想法，但仍不够具体

              所以我大一大二时，大抵也就是敲敲算法题，多数时间还是处于躺平的状态
              大二暑期意识到自己可能更倾向于找工作，当时想着平时自己都是用 C++ 在敲题，再加上网上的 “ Java卷爆了，C++大有可为” 的吹嘘，选择 了C++ 后台的方向，于是乎在暑假期间我看了一本叫《 Accelerated C++ 》的书，并结合 《 C++ Primer》，跳进了 C++ 的坑

              然后呢？然后我发现这条路好像走不太通，先不说自己能不能玩明白 C++
              我好像并不是很喜欢这东西，而且 C++ 后台？哪钻出来的方向？

              我意识到这件事时，已经是大三上差不多过了快一个月的时候了，那时候每天都很迷茫

              但人吧，总有想通的那一天，为啥要从语言出发呢？
              我想从事什么样的工作？互联网方向的
              前端还是后端？后端

              那不就得了，选择 Java 就完事

              互联网寒冬？Java 卷？
              那我就更加努力好了

              到今天，学 Java 也差不多快三个月了，要学的东西还有很多很多
              但我的内心却不再感到迷茫

              # 迈入2024

              在家的时候，跨年的瞬间我大多都是在洗澡之中度过，洗了一年的澡，不过对于每天都要洗澡的我来说倒也差不多
              但由于学校的浴室11点就停止供水，所以倒也不能再在洗澡中跨年了

              2023跨年的时候，我应该是吹完了头发，然后坐着在发呆
              舍友的一句，新年快乐，让我意识到，已经2024了啊
              和舍友们互道了一句新年快乐，给朋友们所在的小群里发一句新年快乐
              在某二次元群里抢点红包（当天qq还出现了吞红包的情况），看着群友们的活跃，让我这个几乎都是一个人跨年的家伙也感到了一点跨年的气氛

              2024的话，要面临的挑战很多，但随之而来的也有着机遇
              倒不如说，如同21年一般，又一次的机会摆在了我的面前，是如同当初仅会幻想的我一般什么也抓不住呢
              还是，认清现实后，燃烧自己呢

              用2024这一年，告诉自己吧
